Some general points

  1. Python is case sensetive. i.e., Upper and lower class characters are treated differently .
  2. All functions or keywords starts with small letters except True, False, and None .
  3. Pass keyword is used for skipping a part of program .
  4. Python is relatively slow compared to other languages .
  5. Syntax should be kept in mind .
  6. Indentation should be proper .
  7. Python is a dynamically typed language .
    • Dynamic typing - When you use same variable for storing multiple type values like string, integer, list, etc its called dynamic typing and you don't have to tell the computer the type of vairable .
    • Static typing - When you have to tell the computer which type of variable you want to use, its called static typing . You can't use same variable for multiple type of values .
  8. There are two types of data types .
    1. Mapping data type - In this type of data type the data is given in the form of key-value pair . Where key is the position/identity of value
      In python there are two mapping data types dictionary and sets(we will talk about it latter) .
    2. sequence data type - In this type data is written in sequences, like string, list, tuple. In this data types indexing is used to find position of values .
    3. Set comes in both sequence and mapping data types .
  9. Indexing - Tells position of any value in a sequence data type(i.e., list, tuple, string, set) . Think of it as counting from any one side of our data type .
    • When counting from left to right start from 0 .
    • When counting from right to left start from -1 .
  10. Data type conversion .

    To convert a data type to other .

    • Implicit type conversion - Computer does it itself .
    • Explicit type conversion - You do it according to you needs
      • e.g., float to int is mostly done implicitly and int to string is converted explicitly(converting string to int will give a garbage value)
    • The type conversion is dependent on which type is to be converted from which type . e.g., int type can not be converted in list type .
  11. A garbage value means a series of 10-12 digits number which has no meaning for us .
  12. To call a functoin braces is must otherwise you will not be calling it .

    Range function and Subscript method

    • Range is a function and subscripting is a method .
    • Both are used in the form of (start, stop, step) .
      • Start is by default 0 .
      • Stop is by default length of data type .
      • The value of stop is stop-1(where stop is the value you give to your function) .
    • The values are passed in braces in range function while in subscript method square brackets are used .
    • The range function is used in loops whereas subscripts are used in with name of list, string then in square brackets other parts .
    • If we pass only one value in range/subscript function, it will be served as stop(i.e., stop-1) . Then next value will be passed to start then next comes stop. No decimal values will be accepted .
    • The values are seperated by commas in range and by colon in subscript .
    • Syntax
        Don't mind the code just see the syntax .
      1. Range
        #Program to print a table of an integer abc .
abc=5
for xyz in range(1,11):     #(stop)11=(stop-1)10
    print(xyz*abc)
#Output - 5 10 15 20 25 30 35 40 45 50
#The output will come in new line after each
        previous output in this case .
      2. Subscript
        #Program to print a part of a string abc
abc="Hello world"
print(abc[0:6])     #(stop)6=(stop-1)5
#Output - Hello

    Parameters and Arguments

    Parameters are vairables we pass to a fucntion while defining them . The parameters are to be passed while calling them as well

    Parameters

    There are 2 types of parameters .

    • Foraml parameters - The vairables we pass to a fucntion inside braces while defining them .
    • Actual parameters - The values we pass to those variables while calling the functions .

    Arguments

    There are 4 types of arguments .

    The way we write/give/pass/put a variable to a function is how the type of argument is defined .
    1. Positional arguments - Arguments that you starts giving values in a order(left to right) .
    2. Default arguments - Arguments having predefined values . We pass values from right to left in default arguments in order to avoid being overwritten by positional arguments or keyword arguments .
    3. Variable length arguments - A variable used after asterisk(*) symbol. we can pass as many values as we want with just one argument . This kind of argument is not highly used .
    4. Keyword arguments - Using variables before passing values while calling the function .
  • Syntax 
    def table(x, y=1):      #y is default argument, x is positional arguments .
    while y<=10:
        print(x*y)
a=int(input("Enter a number: "))
table(x=a)      #x is keyword here because of the way it's value is passed .
#If a would have been passed like - table(a) then it would have been a positional argument
  • Variables inside braces is called parameters. while only variables are called arguments
    e.g., def table(x)      #Here (x) is a parameter whereas x alone is an argument .
